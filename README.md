# Ng4Framework

## Typings 声明管理器
### 1、全局安装typings声明器
```
npm install typings -g
```
### 2、使用typings.json管理Definition
#### 2.1 初始化指令
```
typings init
```
#### 2.2 安装 Definition
```
typings install dt~xxxx --global --save
```
#### 2.2 常用库
```
typings install dt~core-js --global --save
typings install dt~node --global --save
```
