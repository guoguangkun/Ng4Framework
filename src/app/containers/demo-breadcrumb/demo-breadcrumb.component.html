面包屑结构简单，支持自定义分隔符。

API 选项

bg-breadcrumb

面包屑组。

属性 类型 描述 menus Array 面包屑数组 separator String 自定义分隔符 icon String 使用FontAwesome字体图标来自定义分隔符

Default

<bg-breadcrumb [menus]="menus"></bg-breadcrumb>

Separator

<bg-breadcrumb separator="|" [menus]="menus"></bg-breadcrumb>

Icon

<bg-breadcrumb icon="tag" [menus]="menus"></bg-breadcrumb>
