标签页可以轻松的切换应用的不同视图或功能方面，或者浏览分类数据集。

  • 多种主题风格

API 选项

bg-tab-group

必选，表示一组Tab。

属性 类型 描述 direction String 是否增加复选框，默认为false

bg-tab

一个Tab。

属性 类型 描述 header String 标签标题，可选

Default

标签1 标签2 <bg-tab-group> <bg-tab header="标题1"> 标签1 </bg-tab> <bg-tab header="标题2"> 标签2 </bg-tab> </bg-tab-group>

Direction

标签1 标签2 <bg-tab-group direction="right"> <bg-tab header="标题1"> 标签1 </bg-tab> <bg-tab header="标题2"> 标签2 </bg-tab> </bg-tab-group>
