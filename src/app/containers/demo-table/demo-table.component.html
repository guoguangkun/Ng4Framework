数据表可用于显示数据列表。

  • 分页，即时搜索和排序
  • 提供了丰富的支持,复选框

API 选项

bg-table

必选，类似HTML的table，一个bg-table对应一个table。

属性 类型 描述 selection String 是否增加复选框，默认为false pagination String 是否增加分页器，默认为false

bg-table-header

必选，类似HTML的thead，一个bg-table-header对应一个thead。

bg-table-body

必选，类似HTML的tbody，一个bg-table-body对应一个tbody。

bg-table-row

必选，类似HTML的tr，一个bg-table-row对应一个tr。

bg-table-cell

必选，类似HTML的td，一个bg-table-cell对应一个td。

属性 类型 描述 colspan Number 规定单元格可横跨的列数，可选 rowspan Number 规定单元格可横跨的行数，可选
