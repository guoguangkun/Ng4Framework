文件树，简单实用

API 选项

bg-tree

树形菜单。

属性 类型 描述 menus Array 设置树形菜单数据 expanded Boolean 是否展开，默认false。

Default

<bg-tree [menus]="menus"></bg-tree>

Javasript代码：

this.menus = [ {{'{'}} title: '菜单', folder: [ {{'{'}} title: '子菜单', file: [ {{'{'}} title: '孙菜单' {{'}'}} ] {{'}'}} ] {{'}'}}, {{'{'}} title: '菜单', file: [ {{'{'}} title: '子菜单' {{'}'}} ] {{'}'}} ];
