按钮是与应用程序交互的重要元素，它可以由文本或图标组成，并且可以通过各种属性进行增强美化。

  • 图标按钮
  • 圆形按钮
  • 按钮组

API 选项

bg-button-group

可选，按钮组

bButton

必选，用来定义一个按钮

属性 类型 描述 size String 可选，用来设置按钮的尺寸(big,middle(默认),small,mini) color String 可选，用来设置按钮的样式(lightblue、green、yellow、red、blue) disabled Boolean 可选，禁用按钮并阻止他的动作。 默认为false icon String 可选，用来设置图标，默认采取Fontawesome图标 block 无 可选，以块元素展示 circle Boolean 可选，圆角
