工具提示识别元素何时被激活。 它们可能包含有关其功能的简短帮助文本。 例如，它们可能包含有关可操作图标的文本信息。

API 选项

bTooltip

必选，表示当前元素带有提示条

属性 类型 描述 bTooltip String 设置提示框的内容。 tooltipPosition String 设置提示框的位置(top、right、bottom、left)，默认right。

Default

<button bButton bTooltip="提示文字，提示文字，提示文字，哈哈哈">默认(右侧)提示框</button>

Position

<button bButton bTooltip="提示文字，提示文字，提示文字，哈哈哈" tooltipPosition="left">左侧提示框</button> <button bButton bTooltip="提示文字，提示文字，提示文字，哈哈哈" tooltipPosition="top">顶部提示框</button> <button bButton bTooltip="提示文字，提示文字，提示文字，哈哈哈" tooltipPosition="bottom">底部提示框</button>
