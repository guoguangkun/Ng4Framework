Toast是一个微妙的通知，显示在应用程式的内容上。 通常情况下，Toast会在短时间内显示，然后自动关闭。

API 选项

bg-radio

Default

<bg-radio label="男" name="sex" [checked]="true"></bg-radio> <bg-radio label="女" name="sex"></bg-radio>

<bg-radio color="green" label="Theme: green" name="theme" [checked]="true"></bg-radio> <bg-radio color="blue" label="Theme: blue" name="theme"></bg-radio> <bg-radio color="lightblue" label="Theme: lightblue" name="theme"></bg-radio> <bg-radio color="yellow" label="Theme: yellow" name="theme"></bg-radio> <bg-radio color="red" label="Theme: red" name="theme"></bg-radio> <bg-radio color="orange" label="Theme: orange" name="theme"></bg-radio>
